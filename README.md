# Parallel Image Processing System

## Description

A web-based image processing system that compares sequential and parallel image processing methods using Python multiprocessing.

## Technologies

- React
- Flask
- Python
- Multiprocessing
- Pillow

## Features

- Upload Image
- Edge Detection
- Grayscale
- Blur
- Sequential Processing
- Parallel Processing
- Speedup Analysis
